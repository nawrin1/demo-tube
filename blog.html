Discuss the scope of var, let and constant

Var:The scope of var is functional scope. Even if it is declared within a block we can access the var variable from outside the scope as well. Var can be updated and re-declared into the scope.

Let:The scope of let is block scope. Besides, it can be declared without initializing it but the value cannot be accessed. Let can be updated but cannot be re-declared into the scope

Constant:The scope of constant is block scope. Besides, if it is declared in a block it can only be accessed within the block not outside. Constant cannot be updated or re-declared into the scope.

Use cases of null and undefined

Null means a blank value. The value is present and it can be accessed but it is empty. On the other hand,undefined means the value which is not defined that is the variable is declared but no value is assigned to it.. The type of undefined is undefined but the type of null is an object

What do you mean by REST API?

REST API is a way of accessing web services in a simple way without having the hassle of any processing. If a request is made by a client to the server then using REST API the request will be represented and a response will be sent to the client.
